# ANIME WEB APP

This project is a simplified web application inspired by anime webistes like MAL and Anilist, offering basic features to track and explore anime titles.

## Technologies and Libraries
  * React v18
  * Laravel v10
  * TailwindCSS v3
  * Flowbite v1.8
  * React Router v6
  * Apollo Client v3.8
  * Axios v1.5
